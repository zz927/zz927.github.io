原型链详解

​ 构造函数的原型属性(prototype)中，除了拥有构造方法constructor之外，还有一个 proto 属性，该属性指向父亲的原型对象(prototype)。

​ 对象的 proto 属性指向了函数的原型，就构成了原型链。
